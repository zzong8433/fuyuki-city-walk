# 후유키 성배전쟁 (working title: Fuyuki Grail War) — 기획문서 v0.1

## 1. 한 줄 컨셉
후유키 시를 무대로 **7기의 서번트가 벌이는 쿼터뷰 트윈스틱 액션 배틀로얄.**
성배전쟁(7 서번트 → 최후의 1인) = 배틀로얄 구조와 정확히 일치한다.

## 2. 무엇을 차용하나
- **World of Claudecraft** — 권위 서버 멀티플레이, 클래스 스킬 키트, 절차적+CC0 에셋, 웹 배포
- **브롤스타즈** — 빠른 탑다운 전투, 트윈스틱 조작, 짧은 매치, "기본공격 + 궁"
- **이터널 리턴** — 쿼터뷰 시점, 인-매치 성장(마력/아이템), 생존
- **Fate/stay night** — 7 클래스 서번트, **보구(Noble Phantasm)=궁극기**, **영주(Command Seal)**, 마스터

## 3. 무대 — 후유키 시 (지역 = 맵/존)
| 지역 | 성격 | 게임 역할 |
|---|---|---|
| 신토(新都) | 현대 도심·고층빌딩 | 개활 교전, 엄폐물 多 |
| 심산정(深山町) | 구시가 주택가 | 골목·은신(부시) |
| 후유키 교회 | 중립지대 | 세이프존/상점 |
| 류도지(柳洞寺) | 산 위 사원 | 캐스터 거점 모티프 |
| 미온 강·대교 | 두 지역을 잇는 다리 | 병목 교전지 |
| 시민공원 | 성배 강림지 | 최종 수축 지점 |

→ 배틀로얄의 "폭풍" 대신 **성배의 결계가 수축**하는 연출로 대체.

## 4. 7 서번트 클래스 (각: 기본공격 + 스킬 2 + 보구 궁극기)
- **세이버 (Saber)** — 균형 근접. 기본: 광역 검격 / 스킬: 돌진베기·가드(피해감소) / **보구: 약속된 승리의 빛검** — 전방 대형 광선.
- **아처 (Archer)** — 원거리 견제. 기본: 속사 / 스킬: 산탄·재배치 대시 / **보구: 화살비 / 검의 언덕** — 지정 지역 폭격.
- **랜서 (Lancer)** — 고속 일격이탈. 기본: 긴 사거리 창격 / 스킬: 도약 돌진·회피 / **보구: 찌르는 죽음의 창** — 마킹 대상 확정 명중 돌진.
- **라이더 (Rider)** — 기동·돌파. 기본: 사슬 단검 / 스킬: 기마 돌진·속박 / **보구: 천마의 차륜** — 탑승 광역 넉백 돌진.
- **캐스터 (Caster)** — 지역 통제. 기본: 마탄 / 스킬: 사역마 설치·감속 결계 / **보구: 파멸의 빛** — 대형 지속 피해 결계.
- **어쌔신 (Assassin)** — 은신 폭딜. 기본: 단검 / 스킬: 은신(능동)·순간이동 / **보구: 환상의 살수** — 순간이동 일격 콤보.
- **버서커 (Berserker)** — 탱키 광전사. 기본: 강타(느림·강함) / 스킬: 광화(체력↓일수록 피해↑)·도약 강타 / **보구: 열두 시련** — 잠깐 무적/1회 부활.

## 5. Fate 고유 메커닉
- **영주(Command Seal) 3개** — 매치당 3회 한정. 즉시 쿨다운 리셋 / NP 즉시 충전 / 순간 강화 중 택1. 매우 Fate스럽고 전략성↑.
- **마력(Mana)** — 클래스 자원. NP 게이지는 피해·처치로 충전.
- (확장) **마스터/서번트** — 협동 모드에서 마스터가 버프·지원·영주 사용.

## 6. 모드
- **성배전쟁 (FFA 7인 배틀로얄)** — 기본/캐논 모드
- (확장) 2인 1팀 마스터&서번트, 진영전

## 7. 아트 디렉션
- **야간 후유키** — 달, 네온, 안개, 무거운 시네마틱 그레이딩(WoCC 포스트프로세싱 차용)
- 서번트: CC0 모델(KayKit/Quaternius) 스킨 + 보구는 화려한 셰이더 이펙트

## 8. 기술 스택
- 웹 / **Three.js + TypeScript + Vite**, 단일파일 프로토타입 → 점진적 프로젝트화
- 멀티: **Node 권위서버 + WebSocket** (WoCC 패턴 포팅)
- 배포: **Vercel** (현재 워크플로 유지)

## 9. IP 주의 (Type-Moon)
- 원작 IP. **개인/팬용은 무방하나 공개·수익화는 법적 위험.**
- 안전책: **클래스 아키타입 + 오리지널 영령 스킨 + "후유키 영감" 오리지널 지명**으로 시리얼 넘버 제거.

## 10. 로드맵
- **M1 — 테마 프로토타입**: 서번트 2~3종 + 후유키 야간 아레나 1곳 + 영주 메커닉 (싱글/봇)
- **M2**: 7종 전원 + 보구 연출 + 결계 수축 존 + 다지역 맵
- **M3**: 멀티플레이(권위서버) + 매치메이킹
- **M4**: 마스터 시스템 / 협동 모드 / 정식 에셋·사운드
